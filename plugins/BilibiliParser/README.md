# BilibiliParser 插件

NcatBot5 的 B站视频链接解析与下载插件。自动检测聊天消息中的 B 站视频链接（BV号、av号、b23.tv 短链及部分官方小程序卡片），并返回视频的封面、标题、播放量等基础信息。针对群聊环境，它还会尝试直接下载并发送视频本体。

> 本插件代码逻辑改编自 [meowhuan/BilibiliParser](https://github.com/meowhuan/BilibiliParser)，针对 NcatBot5 框架进行了适配与功能重构。


## ✨ 功能特性

- 🔗 **多格式支持**：解析 `BV号`、`av号`、`b23.tv` 短链以及 `CQ:json` 格式的小程序卡片。
- 📊 **信息摘要**：自动爬取并返回视频的封面图片、标题、简介、播放量、点赞、收藏及投币等统计信息。
- 🎥 **自动发视频**：在群聊中触发时，会自动下载视频（基础清晰度）并作为视频文件发送到群内（不支持私聊发视频）。
- 🔐 **扫码登录**：自带二维码扫码登录 B 站功能，获取 Cookies 并使用 `Fernet` 进行本地 AES 强加密存储，有效防拦截防风控。

## 📋 命令与使用方法

### 扫码登录

| 命令 | 说明 | 适用场景 |
| :--- | :--- | :--- |
| `/bparser_login` | 生成 B 站扫码登录二维码，用于获取并保存登录态 | 群聊 / 私聊 |

*注：生成二维码后务必使用 B 站移动端 App 进行扫描授权（3分钟超时）。获取的登录 Cookie 会被自动加密在 `db/` 下。*

### URL / ID 自动解析

当用户直接发送以下类型的内容时，插件会自动响应：
- **BV号/av号**：消息中带有 `BV1xxxxxxxxx` 或 `av12345678` 字样。
- **b23短链**：如 `https://b23.tv/XyZ123` 的分享链接。
- **部分卡片分享**：B站 App 分享到 QQ 的部分 XML/JSON 卡片。

**发送逻辑**：
1. **回复基本信息**：标题、简介、播放点赞等统计信息和封面图。
2. **提取视频直链下载并发送（仅群聊有效）**：紧随基础信息后，发送对应的 MP4 视频文件。若在私聊触发，机器人将提示“当前私聊不允许解析哦~”。

## ⚙️ 技术细节

| 项目 | 说明 |
| :--- | :--- |
| 请求库 | `aiohttp`（异步下载与 API 请求）、`requests`（扫码与二维码轮询） |
| 加密模块 | `cryptography.fernet` |

## ⚠️ 注意事项

1. 视频直接发送可能会导致大额流量消耗或群聊刷屏，建议按需评估。
2. 解析高清（1080P等）视频必须依赖登录信息（通过`/bparser_login`）。默认会请求 `qn=80`（一般对应流畅/高清档）。
