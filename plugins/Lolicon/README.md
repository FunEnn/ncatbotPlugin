# Lolicon 插件

调用 [Lolicon API v2](https://docs.api.lolicon.app/#/setu) 发送随机二次元图片的 NcatBot5 插件。

> **框架要求**：ncatbot5 v5.3.4+

## ✨ 功能特性

- 🖼️ 调用 Lolicon API v2 获取高质量二次元图片
- 💾 本地缓存功能，减少重复下载
- 🔒 R18 内容仅限私聊使用
- 🏷️ 支持单个标签搜索
- ⚡ 并发下载（最多 5 个任务）与分批发送

## 📁 文件结构

```
Lolicon/
├── manifest.toml      # 插件元数据
├── plugin.py           # 插件主逻辑（入口类 LoliconPlugin）
├── cache/              # 图片缓存目录（自动创建）
│   └── cache_index.json
└── requirements.txt
```

## 📋 命令列表

### 用户命令

| 命令 | 说明 | 适用场景 |
| :--- | :--- | :--- |
| `/loli` | 获取 1 张随机图片 | 群聊 / 私聊 |
| `/loli <数量>` | 获取指定数量图片（最多 10 张） | 群聊 / 私聊 |
| `/loli <标签>` | 获取指定标签图片 | 群聊 / 私聊 |
| `/loli <数量> <标签>` | 获取指定数量和标签的图片 | 群聊 / 私聊 |
| `/r18` | 获取 1 张 R18 图片 | **仅私聊** |
| `/r18 <数量> [标签]` | 获取指定数量 R18 图片（最多 5 张） | **仅私聊** |

### 使用示例

```
/loli                    # 获取 1 张随机图片（默认标签：萝莉）
/loli 3                  # 获取 3 张随机图片
/loli 可爱               # 获取 1 张"可爱"标签图片
/loli 5 白丝             # 获取 5 张"白丝"标签图片
```

### 参数解析规则

- 若第一个参数是数字 → 作为数量；第二个参数作为标签
- 若第一个参数不是数字 → 作为标签，数量默认为 1
- 未提供标签时，默认标签为 `萝莉`

## ⚙️ 技术细节

| 项目 | 说明 |
| :--- | :--- |
| API 地址 | `https://api.lolicon.app/setu/v2` |
| 并发下载 | 最多 5 个并发任务 |
| 缓存位置 | `plugins/Lolicon/cache/` |
| 下载超时 | 10 秒 |
| API 超时 | 15 秒 |
| 分批发送 | 每批最多 5 张，批间延迟 0.2 秒 |

## ⚠️ 注意事项

1. R18 内容仅限私聊使用
2. 只支持单个标签搜索
3. 合理使用 API，避免频繁请求
4. 建议定期清理 `cache/` 目录以节省空间
